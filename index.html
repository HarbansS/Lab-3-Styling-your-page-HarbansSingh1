CCS stands for Cascading Styling Sheet. it is used in combination of html to decorate and make your website look better and more presentable. CSS can be used to change anything in your document, color, size, position, font, backgrounds, borders and so much more! To use CSS you can do it in 3 ways, in your opening tag, in the head or in a different document. you need to use {} to add the properties.
